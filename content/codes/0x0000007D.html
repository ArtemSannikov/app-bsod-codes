Кнопка назад

0x0000007D

Шестнадцатеричный код ошибки:

0x0000007D

Название BSoD ошибки:

INSTALL_MORE_MEMORY

Причина:

Код 0x0000007D указывает на недостаток оперативной памяти для загрузки операционной системы Microsoft Windows.

Операционной системы Windows не достаточно оперативной памяти для завершения процесса запуска.

Устранение ошибки:

Увеличьте объем оперативной памяти в компьютере.
